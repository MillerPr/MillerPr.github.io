Challenges

Here are some challenges above and beyond the homework and digital chalkboard tasks.
  1. Create a nav bar that can be loaded on all of your pages. You should create the nav once as a separate html file, then use a script to add this chunk of html to a placeholder on every other page.
  2. Convert all of your events to use event listeners. Your html pages should have no event attributes such as onclick.
  3. Wikipedia API: create anchor links to Wikipedia as results. Use the pageid property from the JSON file. The base URL for loading Wikipedia pages by pageid is the following: https://en.wikipedia.org/?curid= Add the pageid to the end of the URL.
  4. Try the MusicBrainzAPI using both JSON and XML approaches.
