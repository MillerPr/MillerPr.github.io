Tasks

  1. Create the folder structure in VS Code for a website project.
  2. Use an emmet abbreviation to create the basic structure of a new HTML page called index.html.
  3. Demostrate how to use the meta element.
  4. View the HTML file in your localhost.
  5. Create a blank CSS file.
  6. Use an emmet abbreviation to add a link in index.html to your CSS file.
  7. Understand how relative paths work.
  8. Where does href="style.css" find this file?
  9. Where does href="/style.css" find this file?
  10. Where does href="../css/style.css" find this file?
  11. Create a heading in the body of index.html using the h1 element.
  12. In your CSS file, add some style to the h1 heading.
  13. Use Inspect to view your CSS and HTML. Change some CSS using Inspect.
  14. Delete the closing tag of the element. What happens in the DOM?
  15. BONUS: share with us any VS Code extensions or key bindings you are using.

Answers

  1. See the videos for recommendations on folder structure.
  2. The simplest emmet abbreviation is the exclamation point.
  3. The meta element records high-level metadata for the entire page.
  4. The CSS file goes in your CSS directory.
  5. The emmet abbreviation is link:css.
  6. The h1element goes in the body and should contain some text.
  7. The code in your CSS file would be something like: h1 {color:maroon};
  8. The inline CSS style attribure would be something like <h1> style="color:aqua">Hello World!</h1>
