Tasks

  1. Create the folder structure in VS Code for a website project.
  2. Use an emmet abbreviation to create the basic structure of a new HTML page called index.html.
  3. Demostrate how to use the meta element.
  4. Create a blank CSS file.
  5. Use an emmet abbreviation to add a link in index.html to your CSS file.
  6. Create a heading in the body of index.html using the h1 element.
  7. In your CSS file, add some style to the h1 heading.
  8. BONUS: share with us any VS Code extensions or key bindings you are using.

Answers

  1. See the videos for recommendations on folder structure.
  2. The simplest emmet abbreviation is the exclamation point.
  3. The meta element records high-level metadata for the entire page.
  4. The CSS file goes in your CSS directory.
  5. The emmet abbreviation is link:css.
  6. The h1lement goes in the body and should contain some text.
  7. The code would be something like: h1 {color:maroon};
